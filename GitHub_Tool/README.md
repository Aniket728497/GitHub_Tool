# 🔍 GitHub Analyzer (Chrome Extension)

## 📌 Overview

**GitHub Analyzer** is a Chrome extension that enhances the GitHub browsing experience by allowing users to analyze any repository directly from its page.

Once installed, the extension adds an **“Analyze” button** to GitHub repository pages. With a single click, users can generate:

* 📄 A concise summary of the repository
* 📊 A visual graph representing repository insights

---

## 🎯 Purpose

When exploring repositories on GitHub, it can be time-consuming to understand:

* What the project actually does
* How complex or well-structured it is
* To understand each person's contribution in the repository
* Key insights without reading all files

This extension simplifies that process by providing **instant AI-powered analysis directly within GitHub**.

---
*After unpacking this particular extension in chrome extensions: 

* Open any repository page on GitHub

### 1. Analyze Button Appears

* A new **“Analyze” button** is injected into the UI

### 2. Click “Analyze”

* The extension:

  * Extracts repository data
  * Sends it for processing
  * Generates insights

### 3. View Results

* 📄 Repository summary
* 📊 Graphical representation of analysis

## 🚀 Installation Guide

### Load Extension in Chrome

1. Clone the repository:

   ```
   git clone https://github.com/Subhransu27062007/GitHub-Analyzer
   ```

2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project’s extension folder
